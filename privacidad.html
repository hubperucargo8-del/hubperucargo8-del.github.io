Politica de Privacidad

Ultima actualizacion: 21 de mayo de 2026.

Hub Peru Cargo S.A.C., RUC 20602031188, trata los datos personales enviados mediante esta web para atender consultas comerciales, cotizaciones, comunicaciones y registros del Libro de Reclamaciones.

Datos que podemos tratar

  • Datos de identificacion y contacto: nombres, documento, correo, telefono y direccion cuando corresponda.
  • Informacion relacionada con servicios logisticos, cotizaciones, reclamos o quejas.
  • Datos tecnicos basicos de navegacion cuando el usuario acepta cookies analiticas.

Finalidades

Usamos la informacion para responder solicitudes, gestionar servicios, cumplir obligaciones legales, atender reclamaciones y mejorar la seguridad y funcionamiento del sitio.

Base legal y conservacion

El tratamiento se realiza con consentimiento del usuario, ejecucion de solicitudes precontractuales o cumplimiento de obligaciones legales aplicables en Peru. Los datos se conservan durante el plazo necesario para cada finalidad y para atender responsabilidades legales.

Encargados y servicios externos

La web utiliza Google Apps Script y Google Sheets para recibir formularios, y puede usar Google Analytics solo cuando el usuario acepta cookies no esenciales.

Derechos del titular

Puede solicitar acceso, rectificacion, cancelacion u oposicion de sus datos escribiendo a ventas@hubperucargo.com. La solicitud debe permitir validar la identidad del titular.

Seguridad

Aplicamos medidas razonables para proteger la informacion y limitamos la publicacion de codigo o configuraciones internas que no sean necesarias para el funcionamiento de la web.
